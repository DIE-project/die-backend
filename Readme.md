**DIE Backen**

**Author:** [DIE] Organization

**Version:** 1.0.0

**DIE Backen** is a backend application written in Node js to automate the process at [DIE] organization. 

## Installation

Use the package manager [npm](https://www.npmjs.com/) to install the dependencies.

```bash
npm install
```

## Usage

```bash
npm run dev
```
